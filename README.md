# 📝 Less WordPress Theme

> “一直想做一款自己的 WordPress 主题，不需要功能太复杂，简单够用就好。
> 最近有一些时间，就做了这款追求简单轻量的 **Less** 主题。”

## ✨ 核心亮点

* ⚡ **极速性能**：基于 Utility-First 的 **Tailwind CSS** 构建，代码精简，加载速度飞快。
* 🌗 **完美深色模式**：内置系统级深色模式支持。可跟随系统自动切换，或由用户手动切换，提供舒适的夜间阅读体验。
* 📱 **极致响应式**：移动端优先设计，在手机、平板和桌面端均有完美的视觉表现。
* 🎨 **纯净设计**：无冗余元素，专注于内容展示，排版考究，阅读体验极佳。
* 🚀 **本地化优化**：移除对 CDN 的依赖（如 Font Awesome），全线使用本地 SVG 图标，确保国内访问速度与稳定性。

---

## 🛠️ 主要功能

1.  🔍 **SEO 优化**：内置基础 SEO 设置，支持自定义首页标题、关键词和描述。
2.  🔄 **AJAX 加载**：列表页支持 AJAX 点击加载更多文章，无需刷新页面，体验丝滑。
3.  🔗 **社交媒体集成**：页脚支持丰富的社交媒体图标链接（微信、QQ、微博、GitHub、X/Twitter 等）。
4.  🧩 **特色小工具**：内置热门文章、最新文章、热门标签、图片广告等实用小工具，并均支持**侧边栏随动**（Sticky）功能。
5.  💻 **自定义代码**：支持在后台直接添加页头/页脚代码和自定义 CSS，方便接入统计脚本或微调样式。
6.  📊 **免插件计数**：原生支持点赞数 👍、阅读数 👁️ 统计，无需安装额外插件。

## ⚙️ 主题设置指南

启用主题后，请前往 WordPress 后台 **“外观” > “Less 设置”** 进行配置。

### 0. 🛠️ 开发构建（可选）

如果您需要修改主题的样式文件，请按照以下步骤重新编译 Tailwind CSS：

1. 确保已安装 [Node.js](https://nodejs.org/)。
2. 进入主题目录：`cd wp-content/themes/Less`
3. 安装依赖：`npm install`
4. 开启监听模式（开发时）：`npm run watch`
5. 构建生产版本（发布前）：`npm run build`

### 1. 🖼️ 基础设置
* **浅色模式 Logo**：上传适用于白色背景的 Logo。
* **深色模式 Logo**：上传适用于深色背景的 Logo（主题会自动根据模式切换）。
* **Favicon**：支持 jpg、png 格式。

### 2. 🎞️ 幻灯片设置
* 支持幻灯片位置显示开关。
* 支持幻灯片自动切换时间设置（秒）。
* 支持配置 **3 张幻灯片** 及 **2 张推荐卡片**。

### 3. 🏷️ SEO 设置
* 自动将 Tag 设置为 keywords。
* 自动获取文章摘要作为 description。
* 自定义首页 Title、Keywords、Description。

### 4. 🦶 Footer 设置
* 可自定义底部版权信息。
* 可选择显示的社交媒体图标。
* 支持配置链接和二维码图片。

### 5. 🎨 显示设置
* **颜色模式**：提供浅色、暗色、自动三种选项。
* **列表设置**：自定义列表文章显示数量；支持开启/关闭列表封面图左上角的分类显示。
* **信息开关**：作者名、浏览数、评论数、点赞数显示开关。
* **互动开关**：评论功能开启/关闭。

### 6. 👨‍💻 自定义代码
* 页头代码自定义（Header）。
* 页脚代码自定义（Footer）。
* CSS 样式自定义。

## 📦 安装说明

1.  下载主题压缩包 (`.zip`)。
2.  进入 WordPress 后台，点击 **“外观” > “主题” > “安装主题” > “上传主题”**。
3.  选择压缩包并安装，然后点击 **“启用”**。
4.  前往 **“外观” > “Less 设置”** 完成初始化配置。
